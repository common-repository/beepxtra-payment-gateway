== Description ==
Beepxtra Gateway is a plugin for WooCommerce admins to enable a new way for their customers to pay for their products through our gateway.

== Short Description ==
Enable your customers to pay through the Beepxtra gateway in WooCommerce.

== External Service Usage ==
This plugin relies on external services provided by Beepxtra to function. Below are the details of the services and how the plugin interacts with them:

1. **Beepxtra Payment Gateway**
   - **URL**: https://pay.beepxtra.com/gateway
   - **Usage**: This service is used to process payments through the Beepxtra gateway.
   - **Terms of Use**: https://www.beepxtra.com/en/terms-and-conditions
   - **Privacy Policy**: https://www.beepxtra.com/en/privacy-policy

By using this plugin, you agree to the terms of use and privacy policies of these external services.

== Installation ==
1. Go to the WordPress plugin repository and search for "Beepxtra Gateway."
2. Click "Install Now" to download and install the plugin.
3. Once installed, click "Activate Plugin" to activate it.
4. Navigate to WooCommerce settings and enable the Beepxtra payment option. Ensure you have the WooCommerce plugin installed and activated.

== Frequently Asked Questions ==
= Do I need WooCommerce installed? =
Yes, WooCommerce needs to be installed and activated for Beepxtra Gateway to work.
